# YouTube Music Album Sorter (ytmalbumsorter)

YouTube Music でアルバムを検索した際に、リリース年でソートするユーザースクリプトです。

## 機能

- **リリース年順ソート**: アルバムを検索した場合、リリース年順にソートできます
- **昇順/降順切り替え**: 「古い順」から「新しい順」への切り替えが可能

## 前提条件

以下のいずれかのユーザースクリプトサポート拡張機能をブラウザにインストールしてください：

- [Greasemonkey](https://www.greasespot.net/)
- [Tampermonkey](https://www.tampermonkey.net/)

## インストール方法

### Greasemonkey でインストールする場合

1. `src/youtubemusicsort.user.js` ファイルをダウンロード
2. Greasemonkey の管理画面を開く
3. 「インストール」ボタンをクリック
4. ダウンロードした `youtubemusicsort.user.js` ファイルを選択
5. 「保存」ボタンで完了

### Tampermonkey でインストールする場合

1. Tampermonkey アイコンをクリック
2. 「+」アイコン（新規作成）をクリック
3. `src/youtubemusicsort.user.js` ファイルの内容をすべてコピーして貼り付け
4. 「保存」ボタンをクリック

## 使い方

1. YouTube Music でアルバムを検索
   - 例：https://music.youtube.com/search?q=artistname&type=album

2. 「アルバムデータの取得」ボタンをクリック
   - YoutubeMusicはスクロールに伴って徐々にアルバムデータがダウンロードされるUIです。自動的に10回分スクロールダウンした後にホーム位置に戻ります。この動作で検索対象のアルバムが最大約400件分まで読み込まれます。 

3. 「▶ 年代順にソートする」ボタンをクリック

4. アルバムがリリース年の降順/昇順（新しい順/古い順）にソートされます

## ソート順序の切り替え

- 「▼ 年代順にソート（降順）」をクリックすると、昇順（古い順）に切り替わります
- 「▲ 年代順にソート（昇順）」をクリックすると、降順（新しい順）に切り替わります
- 何度クリックしても順序が切り替わります

## デバッグ

ブラウザの開発者ツール（F12）のコンソールを表示すると、処理の詳細なログが出力されます。
